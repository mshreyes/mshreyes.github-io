Even Functions:


A univariate function f(x) is said to be even provided that

f(x) = f(-x)

Geometrically, such functions are symmetric about the y - axis.


Example : Cosine function

Periodic Function

A function f(x) is said to be periodic if there exists a number T > 0, such that,

f(x + T) = f(x)  

for all x.
The smallest such T is called the period of f(x).

Examples : Sine function and Cosine function with least period 2π.

Note: The constant function f(x) = a is periodic with any period T for all nonzero real numbers R, so there is no concept analogous to the least period for constant functions.

Dirichlet's conditions

Any periodic waveform of period p = 2L can be expressed in a Fourier series provided that:

it has a finite number of discontinuities within the period 2L

it has a finite average value in the period 2L

it has a finite number of minima and maxima

Note : Typically, f(x) will be piecewise defined.

Advantage: The function f(x) can have discontinuities, whereas for defining Taylor series, f(x) must be continuous.
